# Temp1

这个仓库用于整理和备份个人在 AI 工具使用、对话记录归档，以及 Vibe Coding 实验项目中的资料。

## 目录结构

```text
chat-logs/
├── claude/     # 和 Claude 的对话记录备份
├── chatgpt/    # 和 ChatGPT 的对话记录备份
└── other/      # 其他 AI 工具或未分类聊天记录

vibe-coding/
├── project-1/  # Vibe Coding 项目 1
└── project-2/  # Vibe Coding 项目 2
```

## 用途

- 备份和管理不同 AI 助手的聊天记录。
- 保存 Vibe Coding 过程中产生的项目代码、想法和实验成果。
- 作为后续学习、复盘和持续迭代的资料库。

## 使用建议

聊天记录可以按日期、主题或工具名称继续细分，例如：

```text
chat-logs/chatgpt/2026-04-28-codex-github-test.md
```

Vibe Coding 项目可以在对应项目目录中添加独立的 `README.md`，记录项目目标、运行方式和开发过程。
